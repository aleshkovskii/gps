# gps


создание или обновление коорддинат и времени оператора
входные данные в мадуль gps в формате:

        feature = {
            x: 1234.213423111,
            y: 56456.56456111,
            id: id,
            operatorName: id,
            role: "operator"
        }

обновление коорддинат и времени оператора
входные данные в мадуль gps в формате:

     position = {
            attributes: {id: id},
            geometry: {
                x: 33323.443434,
                y: 2343.544544
            }
        }
